## VeriGuard AI Landing Page

A modern, high-conversion landing page for **VeriGuard AI** – a cross-platform AI fact-checking assistant that helps seniors and their families identify misinformation on social media and messaging apps.

### Tech stack

- **Framework**: Next.js (App Router, React 18, TypeScript)
- **Styling**: Tailwind CSS with custom `trustBlue` and `safeGreen` palette
- **Icons**: `lucide-react`

### Getting started

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open `http://localhost:3000` in your browser to view the landing page.

### Customization

- Update marketing copy and CTAs in `app/page.tsx`.
- Adjust branding (colors, logo, favicon, metadata) in `tailwind.config.ts`, `app/globals.css`, and `app/layout.tsx`.
